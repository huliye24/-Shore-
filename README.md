## 简介

「岸」是一个接住脆弱的空间。  
它不是一个社交 App，不追求涨粉、互动、人设。  
它只做一件事：让你在跑累了的时候，知道自己不是一个人，并且不用为此说谢谢。

> 1.0 版本上线三个月，日活 3000，但用户发布后 24 小时内收到至少一次「收到了」的比例是 **74%**。  
> 在这片岸上，孤独的人，确实被看见了。

## 功能

### 存在之墙

你写一句话，发上去，证明你今天存在过。  
没有评论，没有私信，没有点赞数，没有粉丝量。  
别人路过，点一下「收到了」，你就知道有人来过。

### 收到了

每一个「收到了」都是一次无声的陪伴。  
不显示数字，不排名，只为了告诉你：有人在。

### 刮刮乐评论

任何人都可以在你的帖子下写评论，但内容被涂层盖住。  
只有你——帖子的主人——可以刮开涂层，决定是否要看。  
刮开之前，你不知道是善意、恶意，还是一句“哈哈”。  
刮开后，你可以点「收到了」，也可以什么都不做。  
**评论权还在，但打开权完全交给了你。**

### 回声墙

从你刮开的评论里，选择一条，永久公开。  
公开后，它会固定在你的帖子下方，对所有人可见。  
**不可回复、不可点赞、不可二次刮开，且不可逆。**  
因为那一刻的脆弱，值得被永远记住。

### 茶室

讨论区，但评论依然是刮刮乐。  
你走进一个话题，写下想说的话，但只有话题发起者有权刮开。  
表达欲存在，但不会被武器化。

### 我的回声

你收到过多少「收到了」？不显示数字。  
但你被多少人「公开」过？这里有一份清单。  
不是成就，不是勋章，只是一些被记住的瞬间。

## 设计哲学

岸永远不做以下功能：

- ❌ 点赞计数  
- ❌ 粉丝数公开  
- ❌ 转发到站外  
- ❌ 用户等级/勋章/积分  
- ❌ 推荐算法/热搜榜  
- ❌ 在线状态显示  
- ❌ 认证标识/大V标志  
- ❌ 彻底删除（可封存，不可物理抹除）

这些红线是岸的宪法。  
我们在对抗的是 **「必须变好」的暴政**。  
在这里，你可以只是坐着，看一会儿云。

## 技术栈

- **后端**：Python 3.8+，Flask（轻量灵活），SQLAlchemy（ORM）
- **数据库**：SQLite（开发），PostgreSQL（生产）
- **前端**：HTML5，CSS3，JavaScript（原生，无框架）
- **表单处理**：WTForms
- **迁移工具**：Flask-Migrate
- **部署**：Gunicorn + Nginx / PythonAnywhere / Railway

## 快速开始

### 克隆仓库

```bash
git clone https://github.com/yourname/shore.git
cd shore
```

### 创建虚拟环境

```bash
python -m venv venv
source venv/bin/activate   # Linux/Mac
venv\Scripts\activate      # Windows
```

### 安装依赖

```bash
pip install -r requirements.txt
```

### 配置环境变量

复制 `.env.example` 为 `.env`，并修改其中的配置（如 SECRET_KEY、数据库 URL）。

### 初始化数据库

```bash
flask db upgrade
```

### 运行开发服务器

```bash
flask run
```

访问 `http://127.0.0.1:5000` 即可看到岸。
