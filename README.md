## 简介

「岸」是一个主打「情绪安全表达」与「轻量匿名社交」的应用。

它不是一个传统社交 App，不追求涨粉、互动、人设。  
它只做一件事：让你在跑累了的时候，知道自己不是一个人，并且不用为此说谢谢。

> Slogan：**不需要变好，只需要坐下**

---

## 功能特性

### 📝 此刻（情绪记录）
- 选择心情标签（平静、开心、难过、生气、爱意、创作欲）
- 记录当下的情绪和想法
- 支持公开到广场或仅自己可见

### 🏛️ 广场（匿名广场）
- 浏览他人的匿名动态
- 打招呼添加好友
- 支持屏蔽敏感词

### 🍵 茶室（兴趣圈子）
- 加入不同的兴趣圈子（深夜树洞、创作者角落、治愈系等）
- 在圈子里发布话题和互动

### 🎫 刮刮乐
- 刮开涂层查看陌生人的留言
- 选择"收到了"或"飘走"
- 发现共鸣的瞬间

### 💬 聊天（私聊功能）
- 与广场认识的好友私聊
- 支持匿名身份和真身身份
- 好友申请管理

### 👤 我的（个人中心）
- 双身份系统：真身 + 匿名身份
- 个人资料管理
- 发布的历史动态

## 快速开始

### 1. 克隆仓库

```bash
git clone https://github.com/huliye24/-Shore-.git
cd -Shore-
```

### 2. 创建虚拟环境

```bash
python -m venv venv
```

**Windows:**
```bash
venv\Scripts\activate
```

**Linux/Mac:**
```bash
source venv/bin/activate
```

### 3. 安装依赖

```bash
pip install streamlit
```

### 4. 运行应用

```bash
streamlit run app_shore_v2_2.py
```

访问 `http://localhost:8501` 即可看到岸。
